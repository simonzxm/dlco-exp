\section{实验内容}

利用 8 位移位寄存器来实现简单的随机数发生器，原理为

\begin{equation}
    x_8=x_4\oplus x_3\oplus x_2\oplus x_0
\end{equation}

\section{思考题}

\textbf{生成的伪随机数序列仍然有一定的规律，如何能够生成更加复杂的伪随机数序列？}

可以考虑以下方式

\begin{enumerate}
    \item 增加寄存器的位数
    \item 选择更加复杂的组合逻辑
    \item 基于环境因素初始化种子
\end{enumerate}
